# Deploy no Railway (Mais fácil e rápido!)

## Setup (3 minutos):

1. **Cadastre-se:** https://railway.app/ (use GitHub)

2. **Novo projeto:**
   - Clique em "New Project"
   - Selecione "Deploy from GitHub repo"
   - Escolha este repositório
   - Selecione a pasta `api`

3. **Configure:**
   - Railway detecta Python automaticamente
   - Variáveis: nenhuma necessária
   - Deploy automático!

4. **Pegue a URL:**
   - Na dashboard, copie a URL pública
   - Cole no `predictionService.ts`

## Vantagens:
✅ URL fixa (não muda)
✅ Sempre online (não precisa deixar Colab aberto)
✅ Deploy automático quando commitar
✅ Grátis: 500h/mês (suficiente)
✅ Mais rápido que ngrok

## Alternativas rápidas:

### Render.com
- https://render.com/
- Mesmo processo
- 750h grátis/mês

### Vercel (só para Next.js/serverless)
- Não suporta Flask diretamente
- Precisaria converter para serverless

**Recomendo Railway - é o mais simples!**
